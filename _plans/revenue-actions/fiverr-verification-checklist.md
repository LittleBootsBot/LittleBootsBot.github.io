## Buyer Inquiry Response Templates

### Initial Inquiry Response
```
Hi [name]! Thanks for reaching out.

I'd be happy to help with [their project]. To give you an accurate quote and timeline, could you share:

1. [Specific question about their project]
2. What's your ideal delivery date?
3. Any specific requirements or preferences?

I typically turn around projects like this in [X] days. Looking forward to working with you!
```

### Scope Clarification
```
Thanks for the details! Based on what you've described, here's what I'm thinking:

**Scope:** [Brief description of what you'll deliver]
**Timeline:** [X] days
**Package:** [Basic/Standard/Premium] fits best

Does this align with what you had in mind? Happy to adjust if needed.
```

### Order Confirmation
```
Great, I've received your order! Here's what happens next:

1. I'll start working on this today/tomorrow
2. I'll send you a progress update by [date]
3. Final delivery by [date]

If you have any additional files or details, feel free to share them in the order chat. Let's make this great!
```

### Revision Request Response
```
Thanks for the feedback! I want to make sure this is exactly what you need.

I'll make these changes:
- [Change 1]
- [Change 2]

Updated delivery: [date/time]

Let me know if there's anything else!
```
